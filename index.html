Jana Mobile

Full Stack Mobile Android Engineering

This summer I interned at Jana, where I implemented features in the MCent Browser Android app to increase time-in-app and retention. This Chromium-based browser rewards users with sim-card top-ups as they browse and trigger ad-impressions, addressing high data-prices in India and other developing markets.

I built a persistent search feature to bring users back to the app and created a UI and API to select interests for our dynamic and customizable news feed system. I also added internal tools to manage user funds and payouts involving international currencies.

EasyStream

Native Android App with Firebase Backend

EasyStream is an app for finding and rating public restrooms. I wrote the initial version with Melanie Belkin using a MySQl and PHP backend. I have since switched to using firebase to dynamically load from a dataset of over 25,000 entries using GeoFire queries. I've also added search and geolocation functionality. We are continuing to monitor the release using Fabric and Firebase Analytics.
